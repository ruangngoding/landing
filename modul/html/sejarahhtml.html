Sejarah HTML

Konsep World Wide Web (WWW) dan Envisaging HTML (1990)

Pada tahun 1989, Tim Berners-Lee di CERN memulai konsep World Wide Web, yang kemudian melibatkan pengembangan HTML sebagai bahasa untuk membuat dan menyusun dokumen yang dapat diakses melalui internet. Pada tahun 1990, ide ini mulai terwujud dengan pengembangan HTML.

HTML 1.0 (1991)

HTML 1.0 diperkenalkan oleh Tim Berners-Lee pada tahun 1991 sebagai versi pertama dari HTML. Meskipun masih sederhana dengan elemen-elemen dasar seperti tautan dan daftar, HTML 1.0 menandai langkah awal dalam pengembangan bahasa markup untuk web.

HTML 2.0 (1995)

Dikeluarkan oleh Internet Engineering Task Force (IETF), HTML 2.0 muncul pada tahun 1995 dan membawa peningkatan dengan menambahkan elemen baru seperti formulir dan skrip, memperkaya kemampuan HTML dalam menciptakan halaman web interaktif.

HTML 3.0 dan Pembentukan W3C (1997)

HTML 3.0, diperkenalkan oleh W3C pada 1997, membawa perkembangan signifikan dengan menambahkan fitur-fitur seperti tabel dan layer untuk tata letak. Pembentukan W3C pada waktu yang sama menjadi langkah penting dalam memastikan perkembangan web sesuai dengan standar yang diterima secara luas.

HTML 4.0 dan Pembagian Menjadi Tiga Varian (1997-1999)

HTML 4.0, dikeluarkan oleh W3C, memperkenalkan sejumlah elemen dan atribut baru. Pembagian menjadi tiga varian (Strict, Transitional, Frameset) memungkinkan adaptasi yang lebih fleksibel sesuai dengan kebutuhan pengembang web.

XHTML 1.0 (2000)

Melangkah ke arah XML yang lebih ketat dalam sintaks, XHTML 1.0 diperkenalkan pada tahun 2000. Ini mengejar kekonsistenan dan interoperabilitas dengan XML, menandai perubahan paradigma dalam penulisan kode HTML.

HTML5 (2008)

HTML5, diumumkan pada 2008, adalah tonggak besar dalam pengembangan web. Mendukung elemen dan atribut baru, multimedia, serta API baru, HTML5 memberikan kemampuan baru untuk menciptakan konten web yang lebih dinamis dan interaktif.

HTML Living Standard (2012 - Sekarang)

Dengan pengumuman HTML Living Standard pada tahun 2012, pendekatan baru diambil dengan menganggap HTML sebagai standar "hidup" yang terus diperbarui tanpa versi resmi. Ini mencerminkan sifat dinamis dan evolusi terus-menerus dari teknologi web. HTML terus berkembang melalui kontribusi komunitas dan adaptasi terhadap kebutuhan web yang berkembang.
