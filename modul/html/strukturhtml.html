Struktur HTML

Dasar HTML (1990)

HTML atau HyperText Markup Language adalah bahasa markup standar yang digunakan untuk membangun dan menyusun konten di web. HTML memberikan struktur dasar bagi sebuah halaman web dan memungkinkan para pengembang untuk menentukan bagaimana kontennya akan ditampilkan di browser.

Berikut adalah pengenalan singkat mengenai dasar atau kerangka HTML:

  1. Elemen HTMLElemen HTML: HTML menggunakan elemen-elemen sebagai blok pembangunan dasar. Elemen-elemen ini ditandai dengan tag, yang biasanya terdiri dari pasangan tag pembuka dan penutup. Contohnya adalah <p> untuk paragraf, <h1> untuk judul level 1, <a> untuk tautan, dan sebagainya.
  2. Struktur Dasar HTML:
    • <!DOCTYPE html>: Ini adalah deklarasi tipe dokumen HTML. Ini memberitahu browser bahwa halaman web menggunakan standar HTML terbaru.
    • <html lang="en">: Tag ini menandai awal dari dokumen HTML dan menyatakan bahwa bahasa yang digunakan dalam dokumen ini adalah bahasa Inggris (en). Isi dari dokumen HTML akan berada di antara tag <html> dan </html>.
    • <head>: Ini adalah bagian dari dokumen HTML yang berisi informasi-informasi terkait dengan halaman web, seperti judul halaman, meta tag, atau tautan ke stylesheet. Tag ini memiliki dua tag anak yaitu <meta> dan <title>.
    • <meta charset="UTF-8">: Ini mendeklarasikan pengaturan karakter dokumen HTML sebagai UTF-8, yang merupakan format karakter yang umum digunakan dan mendukung banyak karakter dari berbagai bahasa.
    • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Ini adalah meta tag yang memberikan instruksi kepada browser terkait tampilan halaman web pada perangkat dengan lebar layar yang berbeda. Pengaturan ini dapat membantu dalam responsifitas desain (responsif terhadap perangkat) dengan mengatur skala awal dan lebar tampilan.
    • <title>Document</title>: Ini adalah judul halaman web yang akan ditampilkan di bilah judul browser. Pengguna dapat melihat judul ini saat mereka menyimpan halaman sebagai favorit atau melihat riwayat browsing.
    • <body>: Ini adalah bagian dari dokumen HTML yang berisi semua konten yang akan ditampilkan di halaman web. Semua teks, gambar, tautan, formulir, dan elemen lainnya akan ditempatkan di antara tag <body.> dan </body>. Pada contoh ini, bagian ini masih kosong, tetapi biasanya inilah tempat utama untuk menempatkan semua elemen visual dan konten halaman web Anda.
  3. AtributAtribut: Elemen HTML dapat memiliki atribut yang memberikan informasi tambahan tentang elemen tersebut. Misalnya, atribut href pada elemen <a> digunakan untuk menentukan URL tautan.
  4. Elemen BersarangElemen Bersarang: Anda dapat menempatkan elemen di dalam elemen lainnya untuk membuat struktur yang lebih kompleks. Misalnya, menempatkan elemen <strong> (teks tebal) di dalam elemen <p> (paragraf).
  5. KomentarKomentar: Anda dapat menyisipkan komentar di dalam kode HTML menggunakan tanda <!-- ini adalah komentar -->. Komentar tidak akan ditampilkan di browser dan berguna untuk memberikan penjelasan atau catatan pada kode.
